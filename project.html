Project Title :

Improve profiling Bank Customers Behavior Using Machine Learning

Objective :

The main objective of this project is to profile the bank customers based upon their transactions. Profiling has been done in such a way that it should be

useful for customer and also for banks to acheive higher profits and to reduce the risk

Description :

In banking sector,banks have large data sets which contains customer information and their history of transactions. So that bank need to divide large

data sets into small clusters to be able to analyze these customers behaviour to achieve higher profits. Banks are confronting many challenges like

default predictions,risk management,customer retention for different purpose to achieve higher profitability and reduce the risk. Machine learning

plays an important role in predicting the behaviour of customer based on a certain group of patterns which identify their future strategy,planning on

offering targeted credit products to customers.
